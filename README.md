# Agent Bot project

This project pretends to add new versions of different functionalities generated from a local LLM model

- Backend: Python: FastAPI + Langchain
- LLM: vLLM + solidrust/Mistral-7B-Instruct-v0.3-AWQ
- Frontend: Next.js

## RELEASES
- v0.1 Basic ChatBot with local LLM inference, langchain and Nemoguardrails
